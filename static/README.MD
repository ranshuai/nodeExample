
##开发目录结构

~~~
static
|—— public ------------------------------- 编译之后的文件
|   └── root
|       |
|       ├── images ------------------------------- 图片文件
|       |
|       ├── js ------------------------------- js 文件
|       |
|       ├── site ------------------------------- html模板文件
|       |
|       └── css ------------------------------- 样式文件
|
|—— src  ------------------------------- 源文件路径
|   ├── images ------------------------------- 图片文件
|   |
|   |—— data ------------------------------- 测试数据
|   |
|   |—— helper ------------------------------- handlebars 注册的方法
|   |
|   ├── js ------------------------------- js 文件
|   |   ├── helper ------------------------------- 工具函数库
|   |   │   └──  util.js
|   |   |
|   |   ├── data ------------------------------- 测试模拟数据
|   |   |     └── data.json
|   |   |
|   |   ├── vendors ------------------------------- js组件
|   |   |
|   |   └── page -------------------------------  页面模块
|   |        |——  ( 继续 )
|   |
|   ├── tem ------------------------------- html模板文件
|   |    |—— common ------------------------------- 公用的模板
|   |
|   └── css ------------------------------- 样式文件
|
├── build ------------------------------- webpack构建配置文件
│   └── webpack.base.people.js
|
|—— lib ------------------------------- 第三方插件
│   └── jquery-1.12.4.min.js
│ 
├── node_modules -------------------------------  模块
|
├── nodeList -------------------------------  栏目列表
│ 
└── templates ------------------------------- 页面模板
~~~

# 初始化依赖
```
npm install
```

# 开发
windows:
```
npm run dev
```

# 打包
windows:
```
npm run build
```
windows环境变量：
"scripts": {
    "dev": "set NODE_ENV=development& webpack-dev-server  --progress --hot",
    "build": "node server & webpack ",
  },
